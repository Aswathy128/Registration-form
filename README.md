# Registration-form
A simple registration form built with HTML and CSS

## 🛠️ Technologies Used
- HTML5
- CSS3
- CSS Variables
- CSS Transitions

## ✨ Features
- Form validation using required attribute
- Responsive design
- Clean and professional styling
- Input focus effects

## 📝 Form Fields
- Username
- Email
- Password
- Date of Birth
- Phone Number
- Gender
- City
- Hobbies
- Message
